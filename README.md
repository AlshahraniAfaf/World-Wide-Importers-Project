# World-Wide-Importers-Project

## Project Overview
An interactive Power BI dashboard built to analyze sales performance, profitability, and operational trends using the World Wide Importers dataset.  
The dashboard provides actionable insights into sales trends, product performance, and state-level profitability to support data-driven decision-making.

## Business Questions
- Total Sales & Profit Over Time  
- Sales & Profit by Item  
- Sales & Profit by State  

## Tech Stack
The dashboard was built using the following tools and technologies:
- 📊 Power BI Desktop – Main data visualization platform used for report creation and interactive analytics.
- 📂 Power Query – Data transformation and cleaning layer used to reshape and prepare the World Wide Importers dataset.
- 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and business logic such as sales, profit, and performance metrics.
- 📝 Data Modeling – Relationships established between tables (DimCity, DimCustomer, DimDate, DimEmployee, DimStockItem, FactSale) to enable cross-filtering and accurate analytical insights.
- 📁 File Format – .pbix for development and report sharing.

## Data Source
- World Wide Importers sample dataset  
- Contains sales transactions, customer data, product details 
- Used to simulate real-world business analytics scenarios

## Features
- Sales and profit analysis over time  
- Product-level performance insights  
- State-level sales and profitability trends  
- Interactive filtering and cross-table relationships  
- Dynamic visual reporting

## Insights
- Identify top-performing products  
- Analyze regional profitability  
- Track sales growth and seasonal trends  
- Support strategic business decisions

## File Format
- .pbix – Power BI development file
